# TechnoHacks Cybersecurity Log Analyzer

## Description

This project was developed as part of the TechnoHacks Cybersecurity & Digital Forensics Internship. It is a Python Flask web application that analyzes server log files, detects failed login attempts, and identifies suspicious IP addresses.

## Features

* Upload Log Files
* Detect Failed Login Attempts
* Identify Suspicious IP Addresses
* Generate Security Analysis Reports
* User-Friendly Web Interface
* Real-Time Log Analysis

## Technologies Used

* Python 3
* Flask
* HTML5
* CSS3
* Git & GitHub
* Kali Linux

## Installation

Install Flask:

```bash
sudo apt install python3-flask -y
```

or

```bash
pip install flask
```

## Run the Project

```bash
python3 app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
